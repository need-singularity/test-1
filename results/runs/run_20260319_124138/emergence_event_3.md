## 창발 급등 이벤트 분석

### 핵심 요약

**7.44 시그마 급등**은 극히 이례적인 통계적 이상치입니다. 정상 분포 기준으로 약 **10억 분의 1** 확률에 해당합니다.

---

### 메트릭 해석: `std_curvature = 0.1611`

`std_curvature`는 리만 다양체의 KNN 그래프에서 각 노드의 곡률 분산을 측정합니다.

- 각 노드 곡률: `1.0 - degree(n) / (2*k)`
- `std_curvature = np.std(curvature_array)`

값 **0.1611**은 노드들의 기하학적 속성이 매우 **비균일**하다는 뜻입니다. 즉, 이 후보가 만든 토폴로지에서 일부 영역은 고도로 연결되고 다른 영역은 희박한 — **이질적 구조**가 형성되었습니다.

---

### 급등 원인 분석

3세대에서 이 급등이 발생한 이유:

| 레이어 | 컴포넌트 | 역할 |
|--------|----------|------|
| representation | `riemannian_manifold` | KNN 그래프 + 곡률 계산 |
| reasoning | `geodesic_bifurcation` | 고곡률 분산 이웃에서 분기점 탐지 → ODE 섭동으로 대안 경로 생성 |
| emergence | `lyapunov_bifurcation` | 초기 조건 민감도 측정 (리아푸노프 지수) |
| verification | `shadow_manifold_audit` | 섭동된 그림자 다양체와 비교하여 환각 검출 |
| optimization | `free_energy_annealing` | F = C(K) - T×H(K) 기반 담금질 최적화 |

이 5개 컴포넌트 조합은 **시스템의 최적 아키텍처**입니다. 4회 라운드 모두 동일한 조합이 fitness=1.0에 도달했습니다.

---

### 왜 3세대에서 급등했는가

`emergence_detector`는 **슬라이딩 윈도우(크기 10)** 기반으로 작동하며, `min_generations=3` 이후부터 감지를 시작합니다. 3세대는 감지가 처음 활성화되는 시점이므로:

1. 1-2세대의 랜덤 초기 집단은 낮은 `std_curvature` 값을 가짐
2. 3세대에서 진화 압력(엘리트 보존 20% + 토너먼트 선택)이 이 최적 조합을 발견
3. 기존 평균 대비 **7.44배 표준편차** 급등 → sigma_spike 트리거

---

### 의미

이 이벤트는 **강한 끌개(attractor)**의 존재를 시사합니다. 아키텍처 탐색 공간에서 이 특정 조합이 주변 조합들과 뚜렷하게 다른 행동을 보이며, 진화 알고리즘이 이를 빠르게 수렴해 찾아냅니다. `std_curvature`의 급등은 이 조합이 만드는 토폴로지가 질적으로 다른 구조 — 국소적 복잡성과 전역적 연결성의 공존 — 를 가진다는 신호입니다.
