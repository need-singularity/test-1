## 창발 급등 이벤트 분석

### 이벤트 요약

| 항목 | 값 | 의미 |
|------|-----|------|
| **Generation** | 17 | 진화 17세대 |
| **Metric** | `n_hyperedges` | 하이퍼엣지 수 (다중 노드 연결) |
| **Value** | 91.0 | 91개 하이퍼엣지 생성 |
| **Sigma** | 2.37σ | 2σ 임계값 초과 → 급등 감지 |

---

### 1. 왜 급등이 발생했는가

이 후보의 아키텍처 조합이 핵심입니다:

- **`dynamic_hypergraph`** (표현층): 유클리드 거리 임계값(1.0) 내 이웃 노드를 클러스터링하여 하이퍼엣지 생성
- **`ising_phase_transition`** (창발층): 임계 온도(~2.269) 근처에서 스핀 정렬 → **상전이**
- **`geodesic_bifurcation`** (추론층): 최단 경로가 분기하는 지점 탐색

**Ising 상전이 + dynamic_hypergraph** 조합에서 Ising 모델이 임계점을 통과하면 전역적 스핀 정렬이 발생하고, 이로 인해 포인트 클라우드의 공간 구조가 급격히 재조직됩니다. 결과적으로 임계값 내 이웃 수가 급증하여 하이퍼엣지 91개가 한꺼번에 형성된 것입니다.

### 2. 2.37σ의 의미

Emergence detector는 **최근 10세대의 슬라이딩 윈도우**에서 현재 값을 제외한 평균/표준편차를 계산합니다:

```
sigma = |current_value - mean(previous)| / std(previous)
```

2.37σ는 임계값(2.0σ)을 넘지만 극단적이지는 않습니다. 과거 데이터에서 `free_energy`가 8.7σ까지 치솟은 사례와 비교하면 **중간 강도의 상전이**입니다.

### 3. 이 후보의 전략적 가치

Hall of Fame 데이터와 대조하면 중요한 시사점이 있습니다:

- **최고 fitness(1.0) 후보들**: `riemannian_manifold` + `lyapunov_bifurcation` 조합
- **이 후보의 조합** (`dynamic_hypergraph` + `ising_phase_transition`): fitness 0.62~0.74 범위

**창발 빈도가 높다고 fitness가 높은 것은 아닙니다.** `n_hyperedges` 급등은 구조적 재조직이 활발하다는 신호이지만, 이 조합의 최종 fitness는 최적 아키텍처 대비 낮습니다. 활발한 창발 활동이 반드시 더 나은 추론 품질로 이어지지 않는다는 패턴과 일치합니다.
