## 창발 급등 이벤트 분석

### 이벤트 요약

| 항목 | 값 | 해석 |
|------|-----|------|
| **세대** | 3 (최소 감지 세대) | 데이터 축적 직후 즉시 감지 |
| **메트릭** | `std_curvature` | Riemannian manifold의 곡률 표준편차 |
| **값** | 0.1514 | 최근 윈도우 평균 대비 급등 |
| **시그마** | **4.22σ** | 임계치 2.0σ의 **2배 이상** — 극히 이례적 |

### 4.22σ가 의미하는 것

정규분포 기준 4.22σ는 발생 확률 약 **0.0012%** (10만 번 중 1.2회)입니다. 이 후보의 곡률 분포가 갑자기 극도로 불균일해졌다는 뜻으로, **매니폴드 표면에 급격한 위상 변화가 발생**한 것입니다.

### 후보 컴포넌트 조합 해석

이 후보는 현재 실행에서 **유일하게 생존하는 "지배적 조합"**입니다 (hall of fame 8개 중 8개 동일):

| 레이어 | 컴포넌트 | 역할 |
|--------|----------|------|
| **표현** | `riemannian_manifold` | k-NN 그래프 → 노드별 곡률 계산 (이 이벤트의 진원지) |
| **추론** | `geodesic_bifurcation` | 고곡률 분기점에서 ODE로 3개 경로 탐색 → 최안정 분기 선택 |
| **창발** | `lyapunov_bifurcation` | 섭동 주입 후 Lyapunov 지수 계산 — 양수 = 카오스 = 창발 신호 |
| **검증** | `shadow_manifold_audit` | 그림자 매니폴드 M*와 비교하여 환각 점수 산출 |
| **최적화** | `free_energy_annealing` | 자유에너지 F = C(K) - T·H(K) 담금질, **100% 생존율** |

### 인과 메커니즘 추정

```
riemannian_manifold: 곡률 분포 생성
        ↓
geodesic_bifurcation: 고곡률 노드 15개를 분기점으로 식별 (branch_stability: 0.878)
        ↓
lyapunov_bifurcation: 양의 Lyapunov 지수(0.007) → 카오스 영역 진입
        ↓
std_curvature 급등: 분기 탐색이 매니폴드 곡률을 불균일하게 재배치
        ↓ 
4.22σ 스파이크로 감지됨
```

핵심은 **geodesic_bifurcation이 15개 분기점을 발견**하면서 곡률이 재분배되었고, 동시에 **lyapunov_bifurcation이 카오스 상태(is_chaotic=1.0)를 확인**한 것입니다. 이 두 프로세스의 상호작용이 곡률 표준편차를 급등시켰습니다.

### 맥락적 평가

같은 세대(gen 3)에서 **3건의 스파이크가 동시 발생**했습니다:
- `hallucination_score` — 3.55σ
- `concept` — 3.06σ  
- **`std_curvature` — 4.22σ** (이 이벤트, 가장 강력)

이는 단일 메트릭의 노이즈가 아니라 **다중 메트릭 동시 급등 = 진짜 위상 전이(phase transition)**로 해석됩니다.

### 주의 사항

- **벤치마크 성능은 아직 낮음**: combined 0.207, contradiction 0.0% — 창발 신호가 강하지만 실제 인지 과제 해결 능력과는 아직 괴리가 있습니다.
- **다양성 부재**: 생존 조합이 하나뿐이라 Phase 2(1000 노드) 확장 시 과적합 위험이 있습니다.
- **Gen 3은 최소 감지 세대**: 윈도우가 짧아 통계적 안정성이 제한적입니다. Phase 2에서 재현 여부 확인이 필요합니다.
