10개 모듈 요약

모듈 책임 입력 → 출력
Orchestrator 전체 루프 제어, Phase 전환, 종료 판단 config → 최종 리포트
ArchitectureGenerator 구성요소 조합, 변이, 재조합 구성요소 풀 → 후보 리스트
TopologySimulator 후보 아키텍처의 위상 연산 실행 후보 + 데이터 → 시뮬 결과
FitnessEvaluator 5개 창발 지표 + 벤치마크 통합 점수 시뮬 결과 → fitness 점수
EvolutionEngine 선택, 인과 기반 표적 변이, 교차 점수 + 인과 → 다음 세대
BenchmarkRunner 3종 벤치마크 자동 실행 후보 + 테스트셋 → 점수
ScaleController 적응형 스케일업 판단 fitness 추이 → 스케일 결정
EmergenceDetector 5개 지표 실시간 추적, 급등 이벤트 세대별 지표 → 이벤트 로그
CausalTracer 구성요소↔지표 인과 분석 세대 히스토리 → 인과 그래프
DataManager 외부 데이터 로드, 복합체 변환, 캐싱 원시 데이터 → 심플리셜 복합체
ResultLogger JSONL/MD 기록, 실행 간 누적 전 모듈 데이터 → 파일
