4-3. 종료 조건 (어느 Phase에서든 적용)

성공 종료
• 환각률 < 0.01 (1%)
• 창발률 > 0.8 (80%)
• 3종 벤치마크 모두 > 0.7
→ 세 조건 동시 충족 시
수렴 종료
• 5세대 연속 fitness 개선 < 0.01
• 인과 분석에서 유의미한 레버 없음
→ 더 이상 개선 불가 판단
자원 한계 종료
• 총 실행 시간 > max_hours (기본 48h)
• 메모리 사용 > 80% 시스템 RAM
→ 안전 장치
루프 한계 종료
• Phase 4→2 루프 > max_loops (기본 10)
• 전체 탐색 조합 공간 > 90% 소진
→ 탐색 공간 고갈
